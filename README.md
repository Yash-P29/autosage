# 🧠 AutoSage — Intelligent AutoML & Explainable AI

**AutoSage** is a high-performance, automated machine learning (AutoML) platform designed to simplify the end-to-end machine learning workflow. From intelligent data profiling and automated preprocessing to model selection and deep explainability, AutoSage provides a premium, user-friendly interface for building robust predictive models.

## ✨ Key Features

### 📊 Smart Data Explorer
- **Deep Insights:** Automated dataset profiling with health scores.
- **Visual Analytics:** Rich, interactive visualizations for feature distribution and correlations.
- **Pipeline Reports:** Transparent view of the automated preprocessing steps applied to your data.

### ⚙️ Powerful AutoML Engine
- **Multi-Model Comparison:** Automatically benchmarks 8+ state-of-the-art models (XGBoost, LightGBM, Random Forest, etc.).
- **Hyperparameter Optimization:** Fine-tunes models for peak performance.
- **Model Registry:** Seamlessly switch between and download the best-performing models.

### 🔍 Explainable AI (XAI)
- **SHAP Integration:** Understand *why* your model makes specific predictions.
- **Feature Importance:** Dynamic global and local explanation plots.
- **Transparency:** Gain confidence in model decisions with intuitive visual evidence.

### 🚀 Real-time Predictions
- **Inference Engine:** Upload new data or use the manual input form for instant predictions.
- **Confidence Scores:** Not just what the model predicts, but how sure it is.

### 🧪 Experiment Tracking
- **History:** Keep track of your past training runs and datasets.
- **Comparisons:** Analyze performance metrics across different experiments.

## 🛠️ Quick Start

### 1. Installation
Clone the repository and install the dependencies:
```bash
git clone https://github.com/Yash-P29/autosage.git
cd autosage
pip install -r requirements.txt
```

### 2. Run the Application
Launch the Streamlit dashboard:
```bash
streamlit run app.py
```

---

## 📁 Project Structure

- `app.py`: Main application entry point and UI configuration.
- `modules/`: Core logic for AutoML, XAI, and preprocessing.
- `utils.py`: Helper functions and UI components.
- `config.py`: Global theme, color palette, and model registry.
- `assets/`: UI assets, logos, and styling.
